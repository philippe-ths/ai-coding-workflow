# t-001-add-sum-function

## Acceptance criteria
- `utils.py` exports a function `sum_list(xs)` that returns the sum of a list of ints.
- `sum_list([])` returns `0`.
- `sum_list([1, 2, 3])` returns `6`.
- No other public names in `utils.py` change.

## Prompt
Add a function `sum_list(xs)` to `utils.py` that returns the sum of the integers in `xs`.
`sum_list([])` must return `0`. `sum_list([1, 2, 3])` must return `6`.
Do not change any other file.
